## Getting Started

Install npm first:

```sh
  npm install
```

Run server:

```sh
  nodemon index.js
  ```

Start app:

```sh
  npm start
```

## Using App

To make a call, add a name and press "Copy Your ID button". Pass your id to whoever you want to talk to, they should put it into "Make A Call" input and press "Call". So simple. Id will reset after each call.
